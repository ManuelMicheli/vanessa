# Cartella Immagini

## Hero Background

Per utilizzare l'immagine come sfondo della homepage:

1. Salva l'immagine con il nome: `hero-background.jpg`
2. Posizionala in questa cartella: `public/images/hero-background.jpg`
3. Formato consigliato: JPG o WebP
4. Dimensioni consigliate: 1920x1080px o superiore
5. Ottimizza l'immagine per il web (compressione) per migliorare le performance

L'immagine verrà automaticamente utilizzata come sfondo della sezione Hero con un overlay scuro per garantire la leggibilità del testo.
